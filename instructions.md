
# دليل تشغيل منصة أبو سعيد بالبيانات اللحظية

لجعل المنصة تسحب البيانات مباشرة من حسابك في MetaTrader 5، اتبع الخطوات التالية:

## 1. المتطلبات
- تثبيت لغة **Python 3.10+**.
- تثبيت برنامج **MetaTrader 5** وتسجيل الدخول لحساب (TASI).

## 2. تثبيت المكتبات اللازمة
افتح نافذة الأوامر (CMD) وشغل الأمر التالي:
```bash
pip install MetaTrader5 fastapi uvicorn pandas
```

## 3. تشغيل خادم الربط
قم بحفظ كود `mt5_server.py` الموجود في المشروع وشغله:
```bash
python mt5_server.py
```

## 4. فتح المنصة
عند تشغيل المنصة (الواجهة)، ستلاحظ ظهور علامة **"LIVE MT5"** باللون الأخضر في الشريط العلوي، وهذا يعني أن البيانات التي تراها الآن هي أسعار السوق الحقيقية من حسابك مباشرة.

## ملاحظات هامة:
- تأكد أن الرموز في MT5 هي (2222, 1120, ...) بدون إضافات، أو قم بتعديل مصفوفة `symbols` في كود `mt5_server.py` لتطابق وسيطك (مثلاً: `2222.SE`).
- المنصة تقوم بفلترة "الأسهم الإيجابية" لحظياً بناءً على ما يسحبه الخادم من MT5.
